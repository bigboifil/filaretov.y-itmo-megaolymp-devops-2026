# filaretov.y-itmo-megaolymp-devops-2026

Для выполнения задания я использовал wsl с ubuntu. Одна машина для управления, и вторая - чистый образ для настройки в соответствии с заданием.

### Перед запуском playbook
0) скачиваем ansible на управляющей машине
1) чтобы ansible работал, настроим ssh аутентификацию по rsa ключам. Таким образом мы сможем проводить настройки на целевом сервере.
2) создадим файл inventory.ini с ip целевого сервера. Таким образом ansible будет знать, где выполнять команды. Пример файла есть в репозитории
3) прописываем таски в плэйбук и можно запускать
